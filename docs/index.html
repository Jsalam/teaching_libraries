getQuad

Gets contiguous elements from the one dimensional array situated at the anchor, right, right-below, and below of an anchor element. The anchor element is at the col and row intersection. The remaining three are on the right, right-lower, and lower of the anchor.

getQuad(arr: array, col: number, row: number, matrixWidth: number): array
Parameters
arr (array) the one-dimensional array
col (number) the column index
row (number) the row index
matrixWidth (number) the matrix row length
Returns
array: An array with elements orders as follows: anchor, right, lower-right, lower

getSurroundingOneDim

Gets the surrounding elements of an element situated in a lattice stored in a one dimensional array. The anchor is the element is at the col and row intersection. It regularly retrieves an array with the surrounding elements starting with the one above the anchor (12:00 o'clock). The next ones are in clockwise order (up, upRight, right, downRight, down, leftDown, left, upLeft). If the anchor is by an edge or corner the returned array inclues only those within the lattice boundaries

getSurroundingOneDim(arr: array, col: number, row: number, matrixWidth: number, theArray: array): Object
Parameters
arr (array) the one-dimensional array
col (number) the column index
row (number) the row index
matrixWidth (number) the matrix row length
theArray (array) Optional. The array to store retrieved elements
Returns
Object: An object with surrounding elements labeled in clockwise order
